함수를 만드는 방법

함수 = 자주 사용되는 기능 구현 모듈 플러그인 라이브러리 
            function  함수명(파라미터, ... ){
                // 처리 
                return 값;
            }
        
            함수명 : 변수명과 같이 만든다 , 첫글자는 영문자 시작, 특수문자는 안되지만 예외 $, _
                    카멜케이스 
            함수 => 함수형 클래스를 만들때는 첫글자를 대문자로 쓰는 파스칼 케이스 
            함수를 호출할 일이 없으면 함수명 생략할 수 있음 (anomymous function : 이름 없는 함수)

            function (){ }
        
            파라미터 : 0개이상, 없어도되고 여러개도 사용가능
            다른 개발자가 만든 견본을 보면 많아 4개정도이므로 4이하로 만들려고 노력 
            낱개데이타, 배열, 객체, 함수전달
            callback : 파라미터로 전달되는 함수 
        
            return : 0개 또는 반드시 한개
            return  result , value , 0;  // 오류 
            하이오더 함수 : return 되는 함수
