연산자 

            산술연산자 : +, -, *, /, % : 연산 결과는 숫자

            3으로 나눈 경우 나머지 값의 경우의 수는? 
            0 1 2 
        
             비교연산자 : > < >=  <=  == !=  ===  !==
             연산결과가  true, false

             ==, !=  : 값만을 비교, 데이터 타입은 비교하지 않음 
             ===, !==  : 값 , 데이터 타입을 모두 비교
        
            논리연산자 : &&(and), ||(or), !(not)
                        &,       |, 
            연산의 결과는   true, false

            true &&  true  = true
            true &&  false  = false
            false && true   = false
            false && false  = false

            true  &  true  = true
            true  &  false  = false
            false &  true   = false // 뒷처리 연산을 하지 않음 
            false &  false  = false

            
            true ||  true  = true
            true ||  false  = true
            false || true   = true
            false || false  = false

            true  |  true  = true  // true 이후의 처리를 하지 않음
            true  |  false  = true
            false |  true   = true    
            false |  false  = false

        
            true : 0이 아닌 숫자, 10, -10, 246, 문자열 , 'A' , 'javascript'

            false : 0, undefined, null, NaN(Not a Number)
